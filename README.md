## 🚀 Projects

### [homelab-gitops](https://github.com/OPerezSandoval/homelab-gitops)
GitOps repo managing a single-node **Talos Kubernetes** cluster. **ArgoCD** pulls
from `main` and continuously reconciles -  including a self-managing
ArgoCD app, ApplicationSets, and platform infra apps.
`Kubernetes` · `ArgoCD` · `GitOps` · `Talos`

### [portfolio-iac](https://github.com/OPerezSandoval/portfolio-iac)
**Terraform** for the AWS infrastructure behind omarperezportfolio.com — multi-env
(prod/dev) S3 + CloudFront (OAI), CloudFront Functions & Lambda@Edge URL rewrites,
WAF, and secrets via Secrets Manager.
`Terraform` · `AWS` · `CloudFront` · `Lambda@Edge`

### [medtracker](https://github.com/OPerezSandoval/medtracker)
A Flask medication-logging app, containerized and deployed to Kubernetes cluster through the
`homelab-gitops` repo via separate dev/prod GitHub Actions pipelines - the full
build → GitOps → deploy loop end to end.
`Flask` · `Docker` · `GitHub Actions` · `Kubernetes`

### [ex288-cert](https://github.com/OPerezSandoval/ex288-cert)
Practice questions and study material for the **Red Hat EX288** (OpenShift
Application Development) certification.
`OpenShift` · `Red Hat`
